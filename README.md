# Hunter Pro CRM v5.0

نظام متكامل لإدارة العملاء العقاريين – مع شات Gemini – حملات واتساب – استخراج أرقام – Responsive كامل.

## مميزات النسخة
- دخول يدوي + Google (محاكاة)
- شات أدمن ذكي (Gemini) داخل الـ dashboard
- استخراج أرقام تلقائي من أي نص
- حملات واتساب (صور + فيديو + إرسال)
- Responsive كامل – يعمل على أي جهاز
- كود نظيف – بدون أخطاء – جاهز للرفع على أي مكان
